# 🚗 Smart Vehicle Rental Management System

Smart Vehicle Rental Management System is a Java desktop application developed using **Java Swing** and **Object Oriented Programming (OOP)** concepts.

This system helps to manage vehicle rental operations in an easy and organized way. Users can add vehicles, view available vehicles, create rental bookings, manage customer details, and calculate rental bills.

## 📌 Project Description

The main purpose of this project is to make vehicle rental management simple and efficient. The system allows the user to manage cars and bikes, rent vehicles to customers, check vehicle availability, and generate billing details based on rental days.

This project is created for the **Object Oriented Programming course** and uses important OOP concepts like classes, objects, inheritance, encapsulation, abstraction, and polymorphism.

## ✨ Main Features

- 🚘 Add and manage vehicles
- 🏍️ Support for cars and bikes
- 👤 Add customer details
- 📅 Create rental bookings
- 💰 Calculate total rental bill
- ✅ Check vehicle availability
- 🖥️ User-friendly Java Swing GUI

## ▶️ How to Run

1. Open the project in **NetBeans IDE**.
2. Make sure Java JDK is installed.
3. Open the project folder.
4. Run the `Main.java` file.
5. The dashboard window will appear.
